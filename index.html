Operadores Matemáticos

  • + : Soma (ex: x + 2)
  • - : Subtração (ex: x - 3)
  • * : Multiplicação (ex: 2 * x)
  • / : Divisão (ex: x / 4)
  • ** : Potência (ex: x ** 2)
  • Math.sqrt(x) : Raiz quadrada
  • Math.sin(x) : Seno
  • Math.cos(x) : Cosseno
  • Math.exp(x) : Exponencial
  • Math.log(x) : Logaritmo natural

Exemplos de Funções

  • x**2 - 4
    (Raiz: x = 2 ou x = -2)
  • Math.sin(x) - 0.5
    (Raiz: x ≈ 0.52)
  • Math.exp(x) - 2
    (Raiz: x ≈ 0.693)
  • Math.sqrt(x) - 3
    (Raiz: x = 9)
  • x/2 + 1
    (Raiz: x = -2)
